# 贡献者行为准则

## 我们的承诺

为了营造一个开放友好的环境，作为贡献者和维护者，我们承诺：参与本项目的每个人都不会受到骚扰，无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向。

我们承诺以促进开放、包容、多元化、健康社区的方式行动和互动。

## 我们的准则

有利于营造积极环境的行为示例：

- 对他人展示同理心和善意
- 尊重不同意见和观点
- 给予建设性反馈，也优雅地接受反馈
- 承担责任并向受我们错误影响的人道歉
- 关注的不仅是个人，更是整个社区的福祉

不可接受的行为示例：

- 使用性化语言或 imagery，以及任何形式的性骚扰或挑逗
- 钓鱼、侮辱或贬损的评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人私人信息
- 其他在专业环境下可被合理认定为不当的行为

## 执行职责

社区领导者的职责包括：

- 澄清和执行本行为准则的标准
- 对其认为不当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施

## 执行措施

对于不可接受的行为，社区领导者可采取以下措施：

1. **警告**：社区领导发出警告，说明违规行为及改正要求
2. **临时封禁**：对违规者实施临时互动封禁
3. **永久封禁**：对严重或屡次违规者实施永久互动封禁

## 适用范围

本行为准则适用于所有社区空间，同时当个人在公共空间代表本社区时也适用。

## 执行

辱骂、骚扰或其他不可接受的行为可通过 [GitHub Issue](https://github.com/fanquanpp/FANDEX-web/issues) 私密联系仓库维护者报告。所有投诉都将及时、公正地被审查和调查。

所有社区领导者都有义务尊重隐私和报告者的安全。
