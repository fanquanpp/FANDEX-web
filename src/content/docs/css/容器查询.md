## 1. 容器查询概述

容器查询允许根据父容器的尺寸而非视口尺寸应用样式。

| 特性       | 媒体查询 | 容器查询 |
| ---------- | -------- | -------- |
| 参照物     | 视口     | 父容器   |
| 组件独立性 |          |          |

## 2. 基本用法

```css
.card-container {
  container-type: inline-size;
  container-name: card;
}

@container card (min-width: 400px) {
  .card {
    display: flex;
    gap: 1rem;
  }
}

@container card (min-width: 200px) and (max-width: 399px) {
  .card {
    display: block;
  }
}
```

## 3. container-type

| 值            | 说明                     |
| ------------- | ------------------------ |
| `inline-size` | 按行内尺寸查询（最常用） |
| `size`        | 按宽高查询               |
| `normal`      | 默认，不作为查询容器     |

## 4. 样式查询

```css
.card-container {
  --theme: dark;
}
@container style(--theme: dark) {
  .card {
    background: #1a1a1a;
    color: white;
  }
}
```
