## 1. 死锁检测

PostgreSQL 使用等图（Wait-For Graph）检测死锁，默认每 1 秒检查一次。

```sql
-- 死锁检测间隔
SET deadlock_timeout = '1s';  -- 默认1秒
```

## 2. 死锁日志

```
ERROR:  deadlock detected
DETAIL:  Process 12345 waits for AccessExclusiveLock on relation 12345; blocked by process 12346.
Process 12346 waits for AccessExclusiveLock on relation 12346; blocked by process 12345.
HINT:  See server log for query details.
CONTEXT:  while updating tuple (0,1) in relation "accounts"
```

## 3. 锁等待超时

```sql
-- 设置锁等待超时
SET lock_timeout = '5s';  -- 5秒超时

-- 默认无超时（一直等待）
SET lock_timeout = 0;
```
