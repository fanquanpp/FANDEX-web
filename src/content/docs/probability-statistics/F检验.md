## 1. 两总体方差比的 F 检验

### 1.1 适用条件

两个正态总体 $N(\mu_1, \sigma_1^2)$ 和 $N(\mu_2, \sigma_2^2)$，检验方差是否相等（方差齐性检验）。

### 1.2 检验步骤

**假设**：

- 双侧：$H_0: \sigma_1^2 = \sigma_2^2$，$H_1: \sigma_1^2 \neq \sigma_2^2$
- 右侧：$H_0: \sigma_1^2 \leq \sigma_2^2$，$H_1: \sigma_1^2 > \sigma_2^2$
- 左侧：$H_0: \sigma_1^2 \geq \sigma_2^2$，$H_1: \sigma_1^2 < \sigma_2^2$

**检验统计量**：

$$F = \frac{S_1^2}{S_2^2} \sim F(n_1 - 1, n_2 - 1)$$

> **约定**：在双侧检验中，将较大的样本方差放在分子上，使 $F \geq 1$。

**拒绝域**：

| 检验类型 | 拒绝域                                                      |
| -------- | ----------------------------------------------------------- |
| 双侧     | $F > F_{\alpha/2}(n_1-1, n_2-1)$                            |
| 右侧     | $F > F_\alpha(n_1-1, n_2-1)$                                |
| 左侧     | $F < F_{1-\alpha}(n_1-1, n_2-1) = 1/F_\alpha(n_2-1, n_1-1)$ |

### 1.3 示例

**例题**：甲乙两台机床加工同种零件，分别抽取 10 件和 8 件，测得 $s_1^2 = 0.064$，$s_2^2 = 0.030$。在 $\alpha = 0.05$ 下检验两机床加工精度是否有显著差异。

**解**：

$H_0: \sigma_1^2 = \sigma_2^2$，$H_1: \sigma_1^2 \neq \sigma_2^2$

$$F = \frac{0.064}{0.030} = 2.133$$

$F_{0.025}(9, 7) = 4.82$，$F = 2.133 < 4.82$，不拒绝 $H_0$。

两机床加工精度无显著差异。

### 1.4 F 检验在 t 检验中的作用

在进行双总体 t 检验之前，通常需要先进行 F 检验判断方差是否相等：

- 若 $\sigma_1^2 = \sigma_2^2$，使用等方差 t 检验
- 若 $\sigma_1^2 \neq \sigma_2^2$，使用 Welch t 检验

## 2. 单因素方差分析

### 2.1 问题

比较 $k$ 个正态总体的均值是否相等。

### 2.2 模型

设 $k$ 个正态总体 $N(\mu_i, \sigma^2)$（$i = 1, 2, \cdots, k$），方差相等，从第 $i$ 个总体抽取 $n_i$ 个样本。

**假设**：$H_0: \mu_1 = \mu_2 = \cdots = \mu_k$，$H_1: \mu_i$ 不全相等

### 2.3 方差分析表

| 来源 | 平方和 | 自由度 | 均方                    | F 值                      |
| ---- | ------ | ------ | ----------------------- | ------------------------- |
| 组间 | $S_A$  | $k-1$  | $\bar{S}_A = S_A/(k-1)$ | $F = \bar{S}_A/\bar{S}_E$ |
| 组内 | $S_E$  | $n-k$  | $\bar{S}_E = S_E/(n-k)$ |                           |
| 总和 | $S_T$  | $n-1$  |                         |                           |

其中：

$$S_T = \sum_{i=1}^k \sum_{j=1}^{n_i} (X_{ij} - \bar{X})^2$$

$$S_A = \sum_{i=1}^k n_i(\bar{X}_i - \bar{X})^2$$

$$S_E = S_T - S_A = \sum_{i=1}^k \sum_{j=1}^{n_i} (X_{ij} - \bar{X}_i)^2$$

### 2.4 检验

$$F = \frac{\bar{S}_A}{\bar{S}_E} \sim F(k-1, n-k)$$

拒绝域：$F > F_\alpha(k-1, n-k)$。

### 2.5 示例

**例题**：三种饲料对猪增重（kg）的试验数据如下：

饲料A：51, 40, 43, 48
饲料B：23, 32, 28, 35
饲料C：23, 25, 26, 18

在 $\alpha = 0.05$ 下检验三种饲料效果是否有显著差异。

**解**：

$\bar{X}_A = 45.5$，$\bar{X}_B = 29.5$，$\bar{X}_C = 23$，$\bar{X} = 32.67$

$S_A = 4(45.5 - 32.67)^2 + 4(29.5 - 32.67)^2 + 4(23 - 32.67)^2 = 657.6 + 40.1 + 374.7 = 1072.4$

$S_E = \sum(X_{ij} - \bar{X}_i)^2 = 74 + 82 + 34 = 190$

$$F = \frac{1072.4/2}{190/9} = \frac{536.2}{21.1} = 25.41$$

$F_{0.05}(2, 9) = 4.26$，$F = 25.41 > 4.26$，拒绝 $H_0$。

三种饲料效果有显著差异。

## 3. 方差分析的前提条件

1. **正态性**：各总体服从正态分布
2. **方差齐性**：各总体方差相等
3. **独立性**：各样本相互独立

当方差齐性不满足时，可使用 Welch 方差分析或非参数方法（如 Kruskal-Wallis 检验）。
