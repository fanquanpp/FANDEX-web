## 1. 单总体均值的 t 检验

### 1.1 适用条件

- 总体服从正态分布 $N(\mu, \sigma^2)$，$\sigma$ 未知
- 用样本标准差 $S$ 代替 $\sigma$

### 1.2 检验步骤

**假设**：

- 双侧：$H_0: \mu = \mu_0$，$H_1: \mu \neq \mu_0$
- 右侧：$H_0: \mu \leq \mu_0$，$H_1: \mu > \mu_0$
- 左侧：$H_0: \mu \geq \mu_0$，$H_1: \mu < \mu_0$

**检验统计量**：

$$T = \frac{\bar{X} - \mu_0}{S/\sqrt{n}} \sim t(n-1)$$

**拒绝域**：

| 检验类型 | 拒绝域                      |
| -------- | --------------------------- |
| 双侧     | $\|T\| > t_{\alpha/2}(n-1)$ |
| 右侧     | $T > t_\alpha(n-1)$         |
| 左侧     | $T < -t_\alpha(n-1)$        |

### 1.3 示例

**例题**：某批零件的长度服从正态分布，标准长度为 50mm。现抽取 6 件，测得长度为：49.8, 50.2, 50.1, 49.7, 50.3, 49.9。在 $\alpha = 0.05$ 下检验该批零件长度是否符合标准。

**解**：

$H_0: \mu = 50$，$H_1: \mu \neq 50$

$\bar{x} = 50.0$，$s = 0.234$

$$T = \frac{50.0 - 50}{0.234/\sqrt{6}} = 0$$

$t_{0.025}(5) = 2.571$，$|T| = 0 < 2.571$，不拒绝 $H_0$。

零件长度符合标准。

## 2. 双总体均值差的 t 检验

### 2.1 等方差情形（Student t 检验）

**条件**：$\sigma_1^2 = \sigma_2^2 = \sigma^2$（未知）

**检验统计量**：

$$T = \frac{(\bar{X} - \bar{Y}) - d_0}{S_w\sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim t(n_1 + n_2 - 2)$$

其中 $S_w^2 = \dfrac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1 + n_2 - 2}$。

### 2.2 不等方差情形（Welch t 检验）

**条件**：$\sigma_1^2 \neq \sigma_2^2$（未知）

**检验统计量**：

$$T = \frac{(\bar{X} - \bar{Y}) - d_0}{\sqrt{\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}}}$$

近似服从 $t(\nu)$，其中自由度

$$\nu = \frac{\left(\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}\right)^2}{\frac{(S_1^2/n_1)^2}{n_1-1} + \frac{(S_2^2/n_2)^2}{n_2-1}}$$

### 2.3 示例

**例题**：比较两种肥料对作物产量的影响。肥料A施于 7 块地，平均产量 $\bar{x} = 48.6$，$s_1 = 4.2$；肥料B施于 6 块地，平均产量 $\bar{y} = 44.2$，$s_2 = 3.8$。假设产量服从正态分布且方差相等，在 $\alpha = 0.05$ 下检验两种肥料效果是否有显著差异。

**解**：

$H_0: \mu_1 = \mu_2$，$H_1: \mu_1 \neq \mu_2$

$$S_w^2 = \frac{6 \times 17.64 + 5 \times 14.44}{11} = \frac{105.84 + 72.20}{11} = 16.185$$

$$T = \frac{48.6 - 44.2}{\sqrt{16.185 \times (1/7 + 1/6)}} = \frac{4.4}{\sqrt{16.185 \times 0.310}} = \frac{4.4}{2.239} = 1.965$$

$t_{0.025}(11) = 2.201$，$|T| = 1.965 < 2.201$，不拒绝 $H_0$。

两种肥料效果无显著差异。

## 3. 配对 t 检验

### 3.1 适用条件

当两个样本是配对的（同一组对象在不同条件下的测量），用配对 t 检验。

### 3.2 方法

设 $d_i = X_i - Y_i$（$i = 1, 2, \cdots, n$），检验 $H_0: \mu_d = 0$。

$$T = \frac{\bar{d}}{S_d/\sqrt{n}} \sim t(n-1)$$

其中 $\bar{d} = \dfrac{1}{n}\sum d_i$，$S_d^2 = \dfrac{1}{n-1}\sum(d_i - \bar{d})^2$。

### 3.3 配对 vs 非配对

- **配对设计**：消除个体差异的影响，检验更灵敏
- **非配对设计**：适用于独立样本

### 3.4 示例

**例题**：为比较两种测量方法的差异，对 8 个样品分别用两种方法测量，差值（方法A - 方法B）为：0.3, -0.2, 0.5, 0.1, -0.1, 0.4, 0.2, -0.3。在 $\alpha = 0.05$ 下检验两种方法是否有系统差异。

**解**：

$H_0: \mu_d = 0$，$H_1: \mu_d \neq 0$

$\bar{d} = 0.1125$，$s_d = 0.299$

$$T = \frac{0.1125}{0.299/\sqrt{8}} = \frac{0.1125}{0.1057} = 1.064$$

$t_{0.025}(7) = 2.365$，$|T| = 1.064 < 2.365$，不拒绝 $H_0$。

两种方法无显著系统差异。

## 4. t 检验汇总

| 检验类型           | 条件                         | 检验统计量                                            | 自由度      |
| ------------------ | ---------------------------- | ----------------------------------------------------- | ----------- |
| 单总体             | $\sigma$ 未知                | $\dfrac{\bar{X}-\mu_0}{S/\sqrt{n}}$                   | $n-1$       |
| 双总体（等方差）   | $\sigma_1^2 = \sigma_2^2$    | $\dfrac{\bar{X}-\bar{Y}}{S_w\sqrt{1/n_1+1/n_2}}$      | $n_1+n_2-2$ |
| 双总体（不等方差） | $\sigma_1^2 \neq \sigma_2^2$ | $\dfrac{\bar{X}-\bar{Y}}{\sqrt{S_1^2/n_1+S_2^2/n_2}}$ | Welch 公式  |
| 配对               | 配对设计                     | $\dfrac{\bar{d}}{S_d/\sqrt{n}}$                       | $n-1$       |
