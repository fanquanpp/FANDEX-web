## 1. 单总体方差的 χ² 检验

### 1.1 适用条件

总体 $X \sim N(\mu, \sigma^2)$，检验 $\sigma^2$ 是否等于某给定值。

### 1.2 检验步骤

**假设**：

- 双侧：$H_0: \sigma^2 = \sigma_0^2$，$H_1: \sigma^2 \neq \sigma_0^2$
- 右侧：$H_0: \sigma^2 \leq \sigma_0^2$，$H_1: \sigma^2 > \sigma_0^2$
- 左侧：$H_0: \sigma^2 \geq \sigma_0^2$，$H_1: \sigma^2 < \sigma_0^2$

**检验统计量**：

$$\chi^2 = \frac{(n-1)S^2}{\sigma_0^2} \sim \chi^2(n-1)$$

**拒绝域**：

| 检验类型 | 拒绝域                                                                   |
| -------- | ------------------------------------------------------------------------ |
| 双侧     | $\chi^2 > \chi^2_{\alpha/2}(n-1)$ 或 $\chi^2 < \chi^2_{1-\alpha/2}(n-1)$ |
| 右侧     | $\chi^2 > \chi^2_\alpha(n-1)$                                            |
| 左侧     | $\chi^2 < \chi^2_{1-\alpha}(n-1)$                                        |

### 1.3 示例

**例题**：某零件直径服从正态分布，标准差规定不超过 0.05mm。现抽取 10 件，测得 $s = 0.06$mm。在 $\alpha = 0.05$ 下检验标准差是否超标。

**解**：

$H_0: \sigma^2 \leq 0.0025$，$H_1: \sigma^2 > 0.0025$

$$\chi^2 = \frac{9 \times 0.0036}{0.0025} = 12.96$$

$\chi^2_{0.05}(9) = 16.919$，$\chi^2 = 12.96 < 16.919$，不拒绝 $H_0$。

标准差未显著超标。

## 2. 拟合优度检验

### 2.1 问题

检验总体分布是否与某个理论分布吻合。

### 2.2 方法

设总体 $X$ 的取值分为 $k$ 个类别，理论概率为 $p_1, p_2, \cdots, p_k$（$\sum p_i = 1$），观测频数为 $n_1, n_2, \cdots, n_k$（$\sum n_i = n$），理论频数为 $np_1, np_2, \cdots, np_k$。

**检验统计量**：

$$\chi^2 = \sum_{i=1}^k \frac{(n_i - np_i)^2}{np_i}$$

在 $H_0$ 成立且 $n$ 充分大时，$\chi^2$ 近似服从 $\chi^2(k - 1 - r)$，其中 $r$ 为需要估计的参数个数。

**拒绝域**：$\chi^2 > \chi^2_\alpha(k-1-r)$

### 2.3 使用条件

- 每个类别的理论频数 $np_i \geq 5$
- 若有 $np_i < 5$ 的类别，需合并相邻类别

### 2.4 示例

**例题**：掷一颗骰子 120 次，结果如下：

| 点数 | 1   | 2   | 3   | 4   | 5   | 6   |
| ---- | --- | --- | --- | --- | --- | --- |
| 频数 | 23  | 16  | 18  | 25  | 22  | 16  |

在 $\alpha = 0.05$ 下检验骰子是否均匀。

**解**：$H_0$: 骰子均匀（$p_i = 1/6$），$H_1$: 骰子不均匀。

理论频数 $np_i = 20$。

$$\chi^2 = \frac{(23-20)^2}{20} + \frac{(16-20)^2}{20} + \frac{(18-20)^2}{20} + \frac{(25-20)^2}{20} + \frac{(22-20)^2}{20} + \frac{(16-20)^2}{20}$$

$$= \frac{9}{20} + \frac{16}{20} + \frac{4}{20} + \frac{25}{20} + \frac{4}{20} + \frac{16}{20} = \frac{74}{20} = 3.7$$

$\chi^2_{0.05}(5) = 11.071$，$\chi^2 = 3.7 < 11.071$，不拒绝 $H_0$。

骰子是均匀的。

## 3. 独立性检验

### 3.1 问题

检验两个分类变量是否独立。

### 3.2 方法

设两个分类变量分别有 $r$ 个和 $c$ 个水平，构成 $r \times c$ 列联表。

**假设**：$H_0$: 两变量独立，$H_1$: 两变量不独立

**理论频数**：

$$e_{ij} = \frac{n_{i\cdot} \times n_{\cdot j}}{n}$$

**检验统计量**：

$$\chi^2 = \sum_{i=1}^r \sum_{j=1}^c \frac{(n_{ij} - e_{ij})^2}{e_{ij}}$$

近似服从 $\chi^2((r-1)(c-1))$。

**拒绝域**：$\chi^2 > \chi^2_\alpha((r-1)(c-1))$

### 3.3 示例

**例题**：调查 200 人的性别与是否喜欢某产品的态度：

|      | 喜欢 | 不喜欢 | 合计 |
| ---- | ---- | ------ | ---- |
| 男   | 40   | 60     | 100  |
| 女   | 70   | 30     | 100  |
| 合计 | 110  | 90     | 200  |

在 $\alpha = 0.05$ 下检验性别与态度是否独立。

**解**：

$e_{11} = \dfrac{100 \times 110}{200} = 55$，$e_{12} = 45$，$e_{21} = 55$，$e_{22} = 45$。

$$\chi^2 = \frac{(40-55)^2}{55} + \frac{(60-45)^2}{45} + \frac{(70-55)^2}{55} + \frac{(30-45)^2}{45}$$

$$= \frac{225}{55} + \frac{225}{45} + \frac{225}{55} + \frac{225}{45} = 4.091 + 5 + 4.091 + 5 = 18.182$$

$\chi^2_{0.05}(1) = 3.841$，$\chi^2 = 18.182 > 3.841$，拒绝 $H_0$。

性别与态度不独立，存在显著关联。

## 4. χ² 检验汇总

| 检验类型   | 目的            | 自由度       |
| ---------- | --------------- | ------------ |
| 方差检验   | 检验 $\sigma^2$ | $n-1$        |
| 拟合优度   | 检验分布        | $k-1-r$      |
| 独立性检验 | 检验两变量独立  | $(r-1)(c-1)$ |
