## 1. 手势

```typescript
Text('Tap me')
  .gesture(TapGesture().onAction(() => console.log('Tapped')))

  // 组合手势
  .gesture(GestureGroup(GestureMode.Exclusive, TapGesture(), LongPressGesture(), PanGesture()));
```
