## 1. 自定义组件

```typescript
@Component
export struct MyButton {
  @Prop text: string
  onButtonClick?: () => void

  build() {
    Button(this.text)
      .width(200)
      .height(50)
      .onClick(() => this.onButtonClick?.())
  }
}
```

## 2. 生命周期

| 回调               | 说明         |
| ------------------ | ------------ |
| `aboutToAppear`    | 组件即将出现 |
| `aboutToDisappear` | 组件即将销毁 |
| `onPageShow`       | 页面显示     |
| `onPageHide`       | 页面隐藏     |
| `onBackPress`      | 返回键按下   |
