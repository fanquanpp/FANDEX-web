## 1. Navigation

```typescript
Navigation() {
  NavRouter() {
    NavDestination() {
      Text('Detail Page')
    }
    .title('Detail')
  }
}
.navDestination(this.buildNavDestination)
```

## 2. Router

```typescript
import router from '@ohos.router';

router.pushUrl({ url: 'pages/Detail', params: { id: 1 } });
router.back();
```
