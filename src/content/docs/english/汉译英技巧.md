## 1. 汉译英的基本原则

### 1.1 汉英语言对比

| 差异 | 汉语     | 英语         |
| ---- | -------- | ------------ |
| 主语 | 常省略   | 必须有主语   |
| 时态 | 靠时间词 | 靠动词变形   |
| 被动 | 少用     | 常用         |
| 修饰 | 前置     | 可前可后     |
| 句式 | 短句并列 | 长句从句     |
| 逻辑 | 隐含     | 显性（连词） |
| 重复 | 允许     | 避免重复     |

### 1.2 汉译英的核心难点

| 难点         | 说明                       |
| ------------ | -------------------------- |
| 主语确定     | 中文常省略主语，英文必须有 |
| 时态选择     | 中文无动词时态变化         |
| 连接词添加   | 中文意合，英文形合         |
| 文化转换     | 中国特有概念需适当处理     |
| 避免中式英语 | 避免按中文思维直译         |

## 2. 主语的确定

### 2.1 补充主语

中文常省略主语，翻译时需补充：

| 中文       | 英文                               | 说明    |
| ---------- | ---------------------------------- | ------- |
| 下雨了。   | **It** is raining.                 | 补充 it |
| 很难说。   | **It** is hard to say.             | 补充 it |
| 必须指出…… | **It** must be pointed out that... | 补充 it |
| 众所周知…… | **It** is well known that...       | 补充 it |

### 2.2 选择主语

中文同一句话可用不同主语翻译：

| 中文             | 译文1                                  | 译文2                                             |
| ---------------- | -------------------------------------- | ------------------------------------------------- |
| 中国经济发展很快 | China's economy is developing rapidly. | China is experiencing rapid economic development. |
| 他英语很好       | His English is excellent.              | He is excellent at English.                       |
| 这本书值得一读   | This book is worth reading.            | It is worthwhile to read this book.               |

### 2.3 物称主语

英语常用无生命事物作主语，中文倾向用人称主语：

| 中文                   | 英文                                            |
| ---------------------- | ----------------------------------------------- |
| 他因为下雨没来         | The rain kept him from coming.                  |
| 我突然想到一个主意     | An idea occurred to me.                         |
| 2026年见证了技术的进步 | The year 2026 witnessed technological progress. |

## 3. 时态的选择

### 3.1 根据时间词确定时态

| 中文时间词       | 英文时态       | 示例                                                     |
| ---------------- | -------------- | -------------------------------------------------------- |
| 昨天、上周、刚才 | 一般过去时     | 他**昨天**去了北京 → He **went** to Beijing yesterday.   |
| 已经、了         | 现在完成时     | 我**已经**完成了 → I **have finished**.                  |
| 正在、在         | 现在进行时     | 她**正在**学习 → She **is studying**.                    |
| 将、会、要       | 一般将来时     | 我们**将**讨论这个问题 → We **will discuss** this issue. |
| 一直             | 现在完成进行时 | 他**一直**在努力 → He **has been working** hard.         |
| 无时间词         | 一般现在时     | 地球绕太阳转 → The earth **goes** around the sun.        |

### 3.2 根据语境确定时态

| 中文             | 英文                                      | 时态选择理由            |
| ---------------- | ----------------------------------------- | ----------------------- |
| 他住在上海       | He lives in Shanghai.                     | 事实 → 一般现在时       |
| 他住在上海三年了 | He has lived in Shanghai for three years. | 持续到现在 → 现在完成时 |
| 他以前住在上海   | He used to live in Shanghai.              | 过去习惯 → used to      |

## 4. 连接词的添加

### 4.1 意合转形合

中文靠逻辑隐含连接，英文需要显性连接词：

| 中文                 | 英文                                          | 添加的连接词 |
| -------------------- | --------------------------------------------- | ------------ |
| 天冷了，多穿点       | **Since** it is cold, put on more clothes.    | Since        |
| 努力学习，就能成功   | **If** you study hard, you will succeed.      | If           |
| 他很累，但继续工作   | He was tired, **but** he continued working.   | but          |
| 下雨了，我们待在家里 | **Because** it is raining, we will stay home. | Because      |

### 4.2 常用连接词

| 逻辑关系 | 连接词                                      |
| -------- | ------------------------------------------- |
| 因果     | because, since, as, therefore, consequently |
| 转折     | but, however, nevertheless, yet             |
| 条件     | if, unless, provided that, as long as       |
| 递进     | moreover, furthermore, in addition          |
| 让步     | although, though, even if, despite          |
| 目的     | so that, in order to, for the purpose of    |
| 时间     | when, while, before, after, until           |

## 5. 文化差异处理

### 5.1 中国特有概念

| 中文     | 翻译策略  | 英文                               |
| -------- | --------- | ---------------------------------- |
| 春节     | 音译+解释 | Spring Festival / Chinese New Year |
| 四大发明 | 直译      | the Four Great Inventions          |
| 关系     | 音译+解释 | guanxi (social connections)        |
| 面子     | 音译+解释 | face (social prestige)             |
| 道家     | 音译+解释 | Daoism / Taoism                    |
| 科举     | 意译      | imperial examination system        |
| 中庸     | 意译      | the Doctrine of the Mean           |

### 5.2 成语翻译

| 策略         | 中文     | 英文                                                                |
| ------------ | -------- | ------------------------------------------------------------------- |
| 直译         | 纸老虎   | paper tiger                                                         |
| 意译         | 画蛇添足 | gild the lily / ruin the effect by adding something superfluous     |
| 借用对应习语 | 挥金如土 | spend money like water                                              |
| 直译+解释    | 塞翁失马 | the old man on the frontier lost his horse (a blessing in disguise) |

### 5.3 数字文化差异

| 中文     | 英文                          | 说明                     |
| -------- | ----------------------------- | ------------------------ |
| 九牛一毛 | a drop in the ocean           | 中文用"牛"，英文用"海洋" |
| 一箭双雕 | kill two birds with one stone | 中文用"雕"，英文用"鸟"   |
| 胆小如鼠 | as timid as a rabbit          | 中文用"鼠"，英文用"兔"   |
| 如鱼得水 | like a duck to water          | 中文用"鱼"，英文用"鸭"   |

## 6. 避免中式英语

### 6.1 常见中式英语

| 中式英语           | 正确英语                     | 问题                     |
| ------------------ | ---------------------------- | ------------------------ |
| open the light     | turn on the light            | 动宾搭配错误             |
| very like          | like very much               | 副词位置错误             |
| give you           | here you are                 | 语境表达错误             |
| long time no see   | it's been a long time        | 语法不规范（已约定俗成） |
| play computer      | use the computer             | 动宾搭配错误             |
| my English is poor | my English is not very good  | poor 用法不当            |
| I very agree       | I completely agree           | very 不修饰动词          |
| welcome to again   | you're welcome to come again | 结构错误                 |

### 6.2 词汇层面的中式英语

| 中式思维        | 正确表达                | 说明                      |
| --------------- | ----------------------- | ------------------------- |
| learn knowledge | acquire/gain knowledge  | learn 不与 knowledge 搭配 |
| big rain        | heavy rain              | rain 用 heavy 修饰        |
| high speed      | fast speed / high speed | 视语境而定                |
| expensive price | high price              | price 用 high/low 修饰    |
| crowded traffic | heavy traffic           | traffic 用 heavy 修饰     |
| save time       | save time               | 这个是正确的              |

### 6.3 句式层面的中式英语

| 中式英语                | 正确英语                 | 问题                     |
| ----------------------- | ------------------------ | ------------------------ |
| Although..., but...     | Although... (no but)     | although 和 but 不能连用 |
| Because..., so...       | Because... (no so)       | because 和 so 不能连用   |
| I think it is not good  | I don't think it is good | 否定应转移到主句         |
| The price is very cheap | The price is very low    | price 用 low/high 修饰   |

## 7. 汉译英实战技巧

### 7.1 无主句的翻译

| 中文             | 翻译方法         | 英文                                |
| ---------------- | ---------------- | ----------------------------------- |
| 要注意安全       | 补充主语         | You should pay attention to safety. |
| 据说他出国了     | 用 it 作形式主语 | It is said that he has gone abroad. |
| 必须立即采取行动 | 用被动语态       | Action must be taken immediately.   |
| 只有一个地球     | 用 there be      | There is only one earth.            |

### 7.2 流水句的翻译

中文的流水句需拆分为英文的从句结构：

> 中文：天黑了，路上行人少了，他加快了脚步往家走。
> 英文：**As** it was getting dark and there were fewer people on the street, he quickened his pace toward home.

### 7.3 四字格的翻译

| 中文     | 英文                             |
| -------- | -------------------------------- |
| 精益求精 | constantly strive for perfection |
| 与时俱进 | keep pace with the times         |
| 因地制宜 | adapt to local conditions        |
| 互利共赢 | mutually beneficial and win-win  |
| 改革开放 | reform and opening-up            |

## 8. 翻译练习方法

### 8.1 对比阅读

- 阅读中英双语对照文本
- 分析译者的翻译策略
- 记录好的翻译表达

### 8.2 回译法

1. 将中文译为英文
2. 将英文回译为中文
3. 对比原文和回译文本
4. 分析差异，改进翻译

### 8.3 仿译法

1. 收集优秀译例
2. 分析翻译技巧
3. 模仿翻译类似文本
