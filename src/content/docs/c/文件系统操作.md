## 1. 目录操作

```c
#include <dirent.h>

DIR *dir = opendir(".");
struct dirent *entry;
while ((entry = readdir(dir)) != NULL) {
  printf("%s\n", entry->d_name);
}
closedir(dir);
```

## 2. 文件属性

```c
#include <sys/stat.h>

struct stat st;
stat("file.txt", &st);
printf("Size: %ld\n", st.st_size);
printf("Mode: %o\n", st.st_mode);
```
