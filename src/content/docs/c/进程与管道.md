## 1. fork

```c
pid_t pid = fork();
if (pid == 0) {
  // 子进程
  execlp("ls", "ls", "-l", NULL);
} else {
  // 父进程
  wait(NULL);
}
```

## 2. 管道

```c
int pipefd[2];
pipe(pipefd);

if (fork() == 0) {
  close(pipefd[0]);
  write(pipefd[1], "Hello", 5);
  close(pipefd[1]);
} else {
  close(pipefd[1]);
  char buf[100];
  read(pipefd[0], buf, sizeof(buf));
  close(pipefd[0]);
}
```
