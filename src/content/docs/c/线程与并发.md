## 1. 线程创建

```c
#include <threads.h>

int thread_func(void *arg) {
  printf("Thread running\n");
  return 0;
}

thrd_t thread;
thrd_create(&thread, thread_func, NULL);
thrd_join(thread, NULL);
```

## 2. 互斥锁

```c
mtx_t mutex;
mtx_init(&mutex, mtx_plain);

mtx_lock(&mutex);
// 临界区
mtx_unlock(&mutex);
mtx_destroy(&mutex);
```

## 3. 条件变量

```c
cnd_t cond;
cnd_init(&cond);
cnd_wait(&cond, &mutex);
cnd_signal(&cond);
cnd_broadcast(&cond);
```
