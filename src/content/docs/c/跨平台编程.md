## 1. 平台检测

```c
#ifdef _WIN32
  #include <windows.h>
  #define PATH_SEPARATOR "\\"
#elif defined(__linux__)
  #include <unistd.h>
  #define PATH_SEPARATOR "/"
#elif defined(__APPLE__)
  #include <mach-o/dyld.h>
  #define PATH_SEPARATOR "/"
#endif
```

## 2. 跨平台抽象

```c
#ifdef _WIN32
  #define sleep_ms(ms) Sleep(ms)
#else
  #define sleep_ms(ms) usleep((ms) * 1000)
#endif
```
