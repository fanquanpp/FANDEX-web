## 1. GDB 调试

```bash
gcc -g program.c
gdb ./program

# GDB 命令
break main
run
next
print variable
backtrace
continue
```

## 2. 静态分析

```bash
# clang-tidy
clang-tidy program.c -- -std=c17

# cppcheck
cppcheck --enable=all program.c

# AddressSanitizer
gcc -fsanitize=address -g program.c
```
