## 1. 对齐规则

```c
struct Example {
  char a;    // 1 字节 + 3 字节填充
  int b;     // 4 字节
  short c;   // 2 字节 + 2 字节填充
};
sizeof(struct Example); // 12

// 优化排列
struct Optimized {
  int b;     // 4 字节
  short c;   // 2 字节
  char a;    // 1 字节 + 1 字节填充
};
sizeof(struct Optimized); // 8
```

## 2. alignof/alignas

```c
#include <stdalign.h>
alignas(16) int aligned_var;
printf("Alignment: %zu\n", alignof(int));
```
