## 1. 信号类型

| 信号    | 说明                 |
| ------- | -------------------- |
| SIGINT  | 中断 (Ctrl+C)        |
| SIGTERM | 终止                 |
| SIGSEGV | 段错误               |
| SIGABRT | 异常终止             |
| SIGKILL | 强制终止（不可捕获） |

```c
#include <signal.h>

void handler(int sig) {
  printf("Received signal %d\n", sig);
}

signal(SIGINT, handler);
```
