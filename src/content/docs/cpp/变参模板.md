## 1. 变参模板

```cpp
template<typename... Args>
void print(Args... args) {
  (std::cout << ... << args) << "\n"; // C++17 折叠表达式
}

print("Hello", " ", "World", "!"); // Hello World!
```

## 2. 折叠表达式

```cpp
template<typename... Args>
auto sum(Args... args) {
  return (args + ...); // 右折叠
}

template<typename... Args>
auto all(Args... args) {
  return (... && args); // 左折叠
}
```
