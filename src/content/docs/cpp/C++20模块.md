## 1. 模块定义

```cpp
// math.cppm
export module math;

export int add(int a, int b) { return a + b; }
export int multiply(int a, int b) { return a * b; }

// 非导出符号（模块私有）
int helper() { return 42; }
```

## 2. 使用模块

```cpp
import math;

int main() {
  std::cout << add(1, 2);     //  导出的
  // helper();                 //  未导出
}
```
