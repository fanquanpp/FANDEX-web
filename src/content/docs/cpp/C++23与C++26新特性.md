## 1. C++23 新特性

- `std::expected<T, E>` — 错误处理
- `std::print` — 格式化输出
- `std::flat_map` / `std::flat_set` — 扁平容器
- `std::generator` — 协程生成器
- `if consteval` — 编译期判断
- `std::unreachable()` — 不可达标记
- 多维下标运算符 `operator[]`

## 2. C++26 新特性

- 反射（Reflection）
- 契约（Contracts）
- `std::text_encoding`
- 线程安全引用计数
- SIMD 库
