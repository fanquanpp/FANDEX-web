## 1. 异常安全等级

| 等级     | 说明                 |
| -------- | -------------------- |
| 不抛出   | `noexcept` 保证      |
| 强保证   | 操作成功或状态不变   |
| 基本保证 | 不泄漏资源，对象可用 |
| 无保证   | 不提供任何保证       |

```cpp
class SafeVector {
  std::vector<int> data_;
public:
  void push_back(int value) {
    data_.push_back(value); // vector::push_back 提供强保证
  }

  void swap(SafeVector& other) noexcept {
    data_.swap(other.data_); // noexcept 保证
  }
};
```
