## 1. 命名空间

```cpp
namespace math {
  constexpr double PI = 3.14159265;
  int add(int a, int b) { return a + b; }
}

using math::PI;
using namespace math; // 不推荐在头文件中使用

// 嵌套命名空间
namespace a::b::c { // C++17
  void func() {}
}
```

## 2. 匿名命名空间

```cpp
namespace {
  int internal_var = 42; // 内部链接，等价于 static
}
```
