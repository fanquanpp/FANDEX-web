## 1. 协程

```lua
local co = coroutine.create(function()
  for i = 1, 5 do
    coroutine.yield(i)
  end
end)

print(coroutine.resume(co)) -- true 1
print(coroutine.resume(co)) -- true 2
```

## 2. 生产者-消费者

```lua
function producer()
  return coroutine.wrap(function()
    while true do
      local value = io.read()
      if not value then return end
      coroutine.yield(value)
    end
  end)
end
```
