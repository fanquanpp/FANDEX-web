## 1. 元表

```lua
local vector = {x = 0, y = 0}
local mt = {
  __add = function(a, b) return {x = a.x + b.x, y = a.y + b.y} end,
  __tostring = function(v) return string.format("(%g, %g)", v.x, v.y) end
}
setmetatable(vector, mt)
```

## 2. 元方法

| 元方法       | 说明     |
| ------------ | -------- |
| `__add`      | +        |
| `__sub`      | -        |
| `__mul`      | \*       |
| `__eq`       | ==       |
| `__lt`       | <        |
| `__index`    | 索引访问 |
| `__newindex` | 索引赋值 |
| `__call`     | 函数调用 |
