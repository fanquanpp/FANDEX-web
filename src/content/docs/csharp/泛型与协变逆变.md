## 1. 泛型约束

```csharp
where T : class           // 引用类型
where T : struct          // 值类型
where T : new()           // 无参构造
where T : IEnumerable     // 实现接口
where T : BaseClass       // 继承基类
```

## 2. 协变与逆变

```csharp
interface IEnumerable<out T> { } // 协变
interface IComparer<in T> { }    // 逆变
```
