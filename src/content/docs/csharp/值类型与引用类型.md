## 1. 对比

| 特性   | struct（值类型） | class（引用类型） |
| ------ | ---------------- | ----------------- |
| 存储   | 栈（通常）       | 堆                |
| 赋值   | 复制值           | 复制引用          |
| 继承   | 不支持           | 支持              |
| 默认值 | 零值             | null              |
| 装箱   | 是               | 否                |
| 适合   | 小型数据         | 复杂对象          |

## 2. 装箱与拆箱

```csharp
int x = 42;
object obj = x;        // 装箱：值类型 → 堆上的引用类型
int y = (int)obj;      // 拆箱：引用类型 → 值类型
```

## 3. 何时使用 struct

- 小于 16 字节
- 生命周期短
- 不可变
- 频繁分配/释放

```csharp
public readonly struct Point {
    public double X { get; }
    public double Y { get; }
    public Point(double x, double y) => (X, Y) = (x, y);
}
```
