## 1. 模式匹配

```csharp
// 类型模式
if (obj is string s) Console.WriteLine(s.Length);

// 属性模式
if (person is { Age: >= 18, City: "Beijing" }) { }

// 列表模式（C# 11）
if (arr is [1, 2, .., 5]) { }

// switch 表达式
string label = status switch {
  200 => "OK",
  404 => "Not Found",
  >= 500 => "Server Error",
  _ => "Unknown"
};
```
