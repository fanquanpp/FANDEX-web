## 1. 对齐规则

```go
// Bad: 24 bytes
type Bad struct {
  a bool   // 1 + 7 padding
  b int64  // 8
  c int32  // 4 + 4 padding
}

// Good: 16 bytes
type Good struct {
  b int64  // 8
  c int32  // 4
  a bool   // 1 + 3 padding
}
```
