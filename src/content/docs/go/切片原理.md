## 1. 切片结构

```go
type slice struct {
  array unsafe.Pointer
  len   int
  cap   int
}
```

## 2. 扩容策略

```go
// 旧容量 < 256: 新容量 = 旧容量 * 2
// 旧容量 >= 256: 新容量 = 旧容量 * 1.25 + 192
```

## 3. 常见陷阱

```go
// 切片共享底层数组
a := []int{1, 2, 3, 4, 5}
b := a[1:3]
b[0] = 99 // a[1] 也变成 99

// 使用 copy 避免共享
c := make([]int, len(b))
copy(c, b)
```
