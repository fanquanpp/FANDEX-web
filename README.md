## 项目背景

FANDEX 项目的创立初衷在于：**协助零基础学习者迈出计算机科学学习的第一步**。

众多零基础学习者在接触计算机科学时，常面临不知从何入手、学习路径不清晰、知识体系碎片化等问题。FANDEX 通过系统化的知识组织与前置知识链路，为学习者构建一条从零基础到计算机专业本科毕业生水平的完整学习路径。

**核心理念：**

- 零基础友好：不预设任何编程经验，从环境搭建与 Markdown 写作起步
- AI 辅助学习：鼓励学习者运用 AI 工具（ChatGPT、Claude、Gemini 等任意外部工具）辅助学习
- 学习过程引导：内容编排遵循学习认知规律，模块间通过前置知识自然衔接
- 专业性保障：知识体系覆盖计算机专业本科核心内容，确保学习深度

## 仓库特色

本仓库是 FANDEX 知识体系的**内容基准与线上学习平台**：

- **内容基准**：全部项目的根本内容源，所有文档、索引及数据均以此为唯一蓝本
- **文档完整**：1995 篇文档覆盖 51 个模块，文档体系最为完善
- **零基础友好**：从第一行代码起步，不假设任何编程经验
- **在线查阅**：基于 GitHub Pages 构建，浏览器直接访问
- **交互学习**：内置进度追踪、术语悬浮、交互测验、知识地图、全文搜索等学习辅助功能
- **AI 辅助**：鼓励学习者使用外部 AI 工具辅助学习，文档结构适配 AI 检索与问答
- **完全开源**：所有内容完全开源，允许任何用途（含商业用途），作者不承担任何责任

## 关于 AI 学习

本项目**不内置任何 AI 功能**。项目的核心理念是：在 AI 时代，学习者应当学会运用 AI 工具进行自主学习。

建议方式：概念理解（请求更通俗解释）、代码学习（解释执行流程或生成变体）、疑难解答（多角度解答）、练习生成（生成练习题）、路径规划（推荐学习顺序）、知识关联（解释概念间关联）。

## 快速开始

```bash
# 安装依赖（要求 Node.js 22 LTS）
npm install

# 本地开发（端口 3000）
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 运行单元测试
npm run test

# TypeScript 类型检查
npm run type-check

# 预发布质量检查（需先构建）
npm run qa

# 添加 Changeset（PR 前必做）
npm run changeset
```

> 开发模式下 Pagefind 搜索索引未生成，搜索功能不可用。需 `npm run build` 后 `npm run preview` 才能使用搜索。

## 部署

仓库已配置 GitHub Actions 自动部署（`.github/workflows/ci.yml`），push 到 `main` 分支即自动触发 8 阶段流水线（lint → type-check → unit-test → e2e-test → build → qa-check → lighthouse → deploy），全部通过后自动部署至 GitHub Pages。

**版本发布**：PR 必须附带 `.changeset/*.md` 声明（由 `changeset.yml` 自动检查），合并至 `main` 后维护者执行 `npm run changeset:version` 更新版本号与 `docs/CHANGELOG.md`。
