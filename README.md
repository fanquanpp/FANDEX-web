## 项目背景

FANDEX 项目的创立初衷在于：**协助零基础学习者迈出计算机科学学习的第一步**。

众多零基础学习者在接触计算机科学时，常面临不知从何入手、学习路径不清晰、知识体系碎片化等问题。FANDEX 通过系统化的知识组织与前置知识链路，为学习者构建一条从零基础到计算机专业本科毕业生水平的完整学习路径。

**核心理念：**

- 零基础友好：不预设任何编程经验，从环境搭建与 Markdown 写作起步
- AI 辅助学习：鼓励学习者运用 AI 工具（ChatGPT、Claude、Gemini 等任意外部工具）辅助学习
- 学习过程引导：内容编排遵循学习认知规律，模块间通过前置知识自然衔接
- 专业性保障：知识体系覆盖计算机专业本科核心内容，确保学习深度

## 项目概述

FANDEX-web 是 FANDEX 知识体系的**线上学习平台**与**内容基准仓库**。作为面向零基础学习者的计算机科学自学教程，以网页形式提供完整的在线学习体验。

每篇文档均经过结构化编排，包含概念讲解、代码示例与实践要点。模块之间通过前置知识关系自然衔接：工具链奠定操作基础，开发语言打开编程世界，数据库承载持久化能力，计算机科学构建理论根基，工程与基础设施连接理论与实践，数据技术与 AI 延伸至前沿应用。学习者可按需跳转，亦可循序渐进。

## 关于 AI 学习

本项目**不内置任何 AI 功能**。项目的核心理念是：在 AI 时代，学习者应当学会运用 AI 工具进行自主学习。

**建议的 AI 辅助学习方式：**

| 学习场景 | AI 工具用法示例                                      |
| :------- | :--------------------------------------------------- |
| 概念理解 | 将文档中的概念粘贴到 AI 工具，请求更通俗的解释或类比 |
| 代码学习 | 让 AI 工具解释代码示例的执行流程，或生成变体练习     |
| 疑难解答 | 遇到不理解的内容时，向 AI 工具提问，获取多角度解答   |
| 练习生成 | 请 AI 工具根据当前模块生成练习题，检验学习成果       |
| 路径规划 | 让 AI 工具根据个人情况推荐学习顺序和重点             |
| 知识关联 | 请 AI 工具解释当前概念与其他模块知识之间的关联       |

## 快速开始

```bash
# 安装依赖
npm install

# 本地开发（端口 3000）
npm run dev

# 构建生产版本
npm run build

# 预览构建结果
npm run preview

# 运行测试
npm run test

# 预发布质量检查（需先构建）
npm run qa

# 内容质量审计（源码级）
node scripts/content-audit.mjs
```

> 开发模式下 Pagefind 搜索索引未生成，搜索功能不可用。需 `npm run build` 后 `npm run preview` 才能使用搜索。

## 部署

仓库已配置 GitHub Actions 自动部署（`.github/workflows/deploy.yml`），push 到 `main` 分支即自动构建发布。

Settings -> Pages -> Source 选择 **GitHub Actions** 即可。
